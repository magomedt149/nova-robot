# NOVA 26 — Free AI Agent

NOVA — бесплатный браузерный ИИ-агент Тумсоева для iPhone, Android и компьютера.

## Что умеет

- голосовой чат на русском и английском;
- режим «Учим английский»: 10 слов и фраз в день, русская транскрипция, произношение и голосовая проверка;
- локальная память, заметки, задачи и таймеры;
- бесплатный локальный AI через WebLLM и WebGPU;
- перевод, погода, Wikipedia, карты и поиск;
- компас, анимации NOVA, Тумсоева и лунной кошки;
- установка на главный экран как PWA.

## Публикация на Netlify

1. Подключите этот GitHub-репозиторий в Netlify.
2. Build command оставьте пустой.
3. Publish directory укажите `.`.
4. Нажмите Deploy.

Секретные ключи и платный API для основной версии не нужны. Локальная модель загружается в браузер только после нажатия кнопки мозга; для неё нужен браузер с WebGPU и интернет при первой загрузке.

Подробная инструкция находится в [README.txt](./README.txt).
