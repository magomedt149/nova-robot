NOVA 26 — FREE AI AGENT
Бесплатный локальный мозг + память + инструменты мини-агента

ЧТО ГЛАВНОЕ

- В проекте нет OpenAI API key и нет платных AI-запросов.
- Банковская карта и подписка для работы NOVA не нужны.
- Базовый мини-агент запускается сразу на любом современном браузере.
- Дополнительный локальный ИИ запускается кнопкой 🧠 и работает прямо на устройстве.
- Исходная v24 не изменена: эта папка является отдельной новой версией NOVA 26.

ТЕЛЕФОНЫ И ANDROID

- Сайт и базовый мини-агент открываются на современных Android, iPhone, планшетах
  и компьютерах. Нужна обычная HTTPS-ссылка на опубликованный сайт.
- На Android локальный большой мозг требует Chrome 121+, Android 12+, WebGPU и
  достаточно свободной памяти. На старых или слабых телефонах модель может не
  загрузиться, даже если сайт открывается.
- Если WebGPU или памяти не хватает, NOVA автоматически остаётся в базовом режиме:
  голос, память, задачи, таймеры, расчёты, погода и компас продолжают работать.
- Микрофон и компас зависят от датчиков телефона, разрешений браузера и HTTPS.
- На iPhone для локального WebGPU-мозга нужен Safari 26+; базовый агент работает
  и без локальной модели.

КАК ВКЛЮЧИТЬ ЛОКАЛЬНЫЙ ИИ

1. Опубликуйте ZIP на HTTPS-сайте, например Netlify.
2. Откройте NOVA в Safari 26+ или современном Chrome на совместимом устройстве.
3. Нажмите кнопку 🧠 «Базовый» в строке состояния.
4. Подтвердите загрузку бесплатной локальной модели.
5. Оставьте NOVA открытой и используйте Wi-Fi. Прогресс появится на кнопке.
6. После надписи «Локальный ИИ» можно свободно разговаривать с NOVA.

Модель загружается только по решению пользователя. Это большой файл, поэтому
первый запуск может занять несколько минут. Браузер хранит модель в своём кэше,
но может удалить её при очистке данных или нехватке свободного места.

Локальный мозг использует Qwen2.5 0.5B через WebLLM и WebGPU. Он бесплатный,
работает без облачного AI-сервера и не отправляет разговор платному провайдеру.
Маленькая локальная модель слабее ChatGPT: она иногда ошибается и не должна
использоваться как единственный источник медицинских, юридических или финансовых
решений.

ЕСЛИ ЛОКАЛЬНЫЙ ИИ НЕ ПОДДЕРЖИВАЕТСЯ

NOVA продолжит работать как бесплатный базовый мини-агент. Память, задачи,
таймеры, расчёты, погода, Википедия, голосовые команды, компас и развлечения
не требуют локальной языковой модели.

ПРИВАТНОСТЬ

- Имя, заметки, задачи и история хранятся локально в браузере этого устройства.
- Локальный ИИ обрабатывает диалог на устройстве.
- Для погоды название города отправляется Open-Meteo.
- Для Википедии вопрос отправляется в официальный API Википедии.
- Карты, YouTube и веб-поиск открываются только после прямой команды пользователя.
- NOVA не умеет сама отправлять письма, публиковать посты, покупать товары или
  изменять чужие аккаунты.

ПОГОДА И БЕСПЛАТНЫЕ ЛИМИТЫ

Погода использует Open-Meteo без ключа. Их бесплатный публичный доступ предназначен
для некоммерческого использования и имеет лимиты. Источник указывается в каждом
ответе о погоде. Для коммерческого массового сайта нужно отдельно проверить их
актуальные условия или заменить источник.

КАК ОПУБЛИКОВАТЬ НА NETLIFY

1. Заархивируйте содержимое этой папки так, чтобы index.html находился в корне ZIP.
2. Загрузите ZIP в Netlify Deploys.
3. Откройте опубликованную HTTPS-ссылку.
4. Разрешите микрофон после нажатия на микрофон.
5. Нажмите компас и разрешите движение и ориентацию.
6. Для установки на iPhone откройте «Поделиться» → «На экран Домой».

ВАЖНО

- Микрофон, PWA, WebGPU и компас нельзя полноценно проверить при открытии файла
  index.html прямо из ZIP. Нужен HTTPS-сайт.
- Таймер сработает только пока NOVA открыта; браузер может остановить страницу в фоне.
- Для погоды, Википедии, карт, поиска и первой загрузки модели нужен интернет.
- Базовые заметки, задачи, расчёты и уже загруженный локальный ИИ могут работать
  без платного сервиса.
